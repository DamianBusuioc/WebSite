# Product Showcase Website

This website is built using HTML and CSS (no WordPress), perfect for small businesses or stores that want to showcase their physical products.

## What's included:

- Home Page (index.html)
- About Us Page
- Products Page with 10 products and review sliders
- Contact Page with a functional HTML form
- Responsive design (mobile-friendly)
- Clean and well-organized code

## How to use:

1. Open the `index.html` file in your browser.
2. All files are already linked and functional.

## How to edit:

- Change styles in `style.css`
- Replace images in the `img/` folder
